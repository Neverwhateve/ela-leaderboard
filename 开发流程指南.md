## 🎯 项目概述

### 示例项目
- **项目名称**：ELA 积分榜
- **项目类型**：React + Vite 前端项目
- **功能特点**：
  - 排行榜展示
  - 玩家个人查询
  - 特殊活动页面（如五一劳动节）

- **框架**：React
- **构建工具**：Vite
- **UI 库**：Tailwind CSS + animal-island-ui
- **数据存储**：JSON 文件
- **部署**：Vercel / GitHub Pages

## 🛠️ 开发流程

### 1. 需求分析 & 规划

首先明确你要做什么：

```markdown
- ✅ 创建五一劳动节特别活动页面
- ✅ 添加动态背景
- ✅ 美化 UI 样式
- ✅ 推送到 GitHub
```

### 2. 准备开发环境

确保你的开发环境已就绪：

```bash
# 克隆项目
git clone <repository-url>

# 安装依赖
cd <project-folder>
npm install

# 启动开发服务器
npm run dev
```

### 3. 创建新页面

#### 3.1 创建页面组件

创建独立的页面文件，例如 `src/NewPage.jsx`：

```jsx
import React from 'react';

const NewPage = ({ onBack }) => {
  return (
    <div className="min-h-screen p-4 md:p-8">
      {/* 返回按钮 */}
      <button onClick={onBack}>← 返回</button>
      
      {/* 页面内容 */}
      <h1>新页面</h1>
    </div>
  );
};

export default NewPage;
```

#### 3.2 添加页面切换逻辑

在主文件 `App.jsx` 中添加路由/状态管理：

```jsx
import { useState } from 'react';
import NewPage from './NewPage';

function App() {
  const [currentPage, setCurrentPage] = useState('home');
  
  return (
    <>
      {currentPage === 'newpage' ? (
        <NewPage onBack={() => setCurrentPage('home')} />
      ) : (
        <div>
          {/* 主页内容 */}
          <button onClick={() => setCurrentPage('newpage')}>
            进入新页面
          </button>
        </div>
      )}
    </>
  );
}
```

### 4. 添加动态背景

#### 4.1 下载背景图片

```bash
# 将背景图片放入 public 目录
public/home_bg.webp
```

#### 4.2 添加 CSS 样式

编辑 `src/index.css`：

```css
body {
  /* 动态背景 */
  background: url('/home_bg.webp') 0 0 / auto repeat, #7DC395;
  animation: bgScroll 80s linear infinite;
  position: relative;
  overflow-x: hidden;
}

@keyframes bgScroll {
  0% { background-position: 100% 0%; }
  100% { background-position: 0% 100%; }
}
```

### 5. UI 样式美化

#### 5.1 标题样式

```jsx
<h1 style={{
  fontFamily: "Nunito, 'Zen Maru Gothic', -apple-system, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif",
  fontSize: 'clamp(2.5rem, 5vw, 60px)',
  fontWeight: 800,
  lineHeight: 1.1,
  color: '#FFF9E6',
  textShadow: '0px 4px 1px rgba(0, 0, 0, 0.4)',
}}>
  标题
</h1>
```

#### 5.2 卡片样式

```jsx
<div style={{ backgroundColor: '#f7f3df' }} className="rounded-acnh p-6 shadow-acnh">
  {/* 卡片内容 */}
</div>
```

#### 5.3 文字样式

```jsx
<div style={{
  fontFamily: "Nunito, 'Zen Maru Gothic', -apple-system, 'PingFang SC', 'Hiragino Sans GB', 'Microsoft YaHei', sans-serif",
  fontSize: '17px',
  fontWeight: 500,
  color: '#7c5734',
  lineHeight: 1.7,
}}>
  文字内容
</div>
```

### 6. 数据管理

创建数据文件 `src/data.js`：

```js
// 数据存储
export const data = {
  // 玩家数据
  players: [
    { id: 1, name: '玩家1', score: 100 },
    { id: 2, name: '玩家2', score: 90 }
  ],
  
  // 特殊活动数据
  specialEvent: {
    title: '活动标题',
    description: '活动描述'
  }
};
```

## 💡 实用技巧

### 1. 网站元素参考

当你看到喜欢的网站元素时，可以这样分析：

#### 方法 1：浏览器检查元素
1. 右键点击元素 → "检查"
2. 查看 HTML 结构和 CSS 样式
3. 复制有用的代码

#### 方法 2：保存网站资源
```bash
# 下载网站背景图片
curl -o public/home_bg.webp <image-url>
```

#### 方法 3：查看网站源代码
- 右键 → "查看网页源代码"
- 查看引用的 CSS/JS 文件

### 2. 使用现成组件库

animal-island-ui 是一个很好的选择：

```jsx
import { Button, Modal } from 'animal-island-ui';
import 'animal-island-ui/style';

function App() {
  return <Button>按钮</Button>;
}
```

### 3. CSS 变量配置

在 `index.css` 中定义设计令牌：

```css
:root {
  --animal-primary-color: #19c8b9;
  --animal-text-color: #725d42;
  --animal-bg-color: #f8f8f0;
}

body {
  color: var(--animal-text-color);
  background-color: var(--animal-bg-color);
}
```

### 4. 响应式设计

```jsx
<div className="max-w-4xl mx-auto">
  <div className="grid grid-cols-1 md:grid-cols-2 gap-4">
    {/* 移动端 1 列，桌面端 2 列 */}
  </div>
</div>
```

### 5. 本地预览和调试

#### 实时预览
```bash
npm run dev
```

#### 构建生产版本
```bash
npm run build
```

#### 预览生产构建
```bash
npm run preview
```
